# Нагорный Андрей  
### Тестовое задание для KODE (React-разработчик стажировка)
  


1. **Дизайн и каркас компонентов**
   * Изучить макет и понять концепцию.
   * Сверстать `Search` и `Select` (со слайдером Swiper).
   * Сверстать `UserList` (сначала скелетон загрузки, потом обычные карточки, затем экраны ошибок и пустого поиска).

2. **Работа со стейтом и API**
   * Добавить глобальный стейт и сделать срез (`userSlice`).
   * Изучить документацию и ответы API, написать модуль для Axios.
   * Доработать срез под сетевые запросы.
   * Сделать фильтрацию списка (протестировать варианты на клиенте и через сервер).
   * Сверстать модалку сортировки.

3. **Роутинг и логика дат**
   * Настроить роутинг и создать страницу пользователя (`UserProfile`).
   * Написать преобразование даты рождения.
   * Сделать разделение списка на блоки по годам для дней рождения.

4. **Кэш, Pull-to-Refresh и Offline**
   * Добавить кэширование результатов запросов прямо в стейт.
   * Добавить Pull-to-Refresh с принудительным обновлением.
   * Настроить сохранение состояний при переходах.
   * Добавить в стейт флаг сети и сделать плашки-уведомления (онлайн/оффлайн).

5. **Локализация и финал**
   * Создать файлы языков и перевести тексты в компонентах.
   * Настроить автоматическое определение языка из браузера.
   * Доделать адаптацию под телефоны (Bottom Sheet для модалки).
   * Реализовать темную тему.
   * Провести финальную оптимизацию кода и сжать картинки.
